##RENTA APTOS
Juan David Martinez Castillo
El proyecto consiste en el desarrollo de un sistema informático para la gestión de alquiler de apartamentos. La aplicación permitirá registrar propiedades, administrar información de propietarios e inquilinos, controlar contratos de arrendamiento, gestionar pagos y generar reportes de ocupación e ingresos. Su objetivo es optimizar los procesos de administración de inmuebles, reducir errores en el manejo de la información y facilitar el seguimiento de los alquileres de manera rápida, segura y eficiente.
ESTRUCTURA UTILIZADA: MVC
